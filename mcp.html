MCP Server

Connect the AIVIOS Coding Coach to your AI client. 8 tools that turn your assistant into a coding mentor.

Quick Install

npx aivios-coding-coach

No API keys. No database. No server. Just Node.js 18+ and an internet connection.

Claude Desktop Configuration

Add this to your claude_desktop_config.json:

{
  "mcpServers": {
    "aivios-coding-coach": {
      "command": "npx",
      "args": ["aivios-coding-coach"]
    }
  }
}

Codex Configuration

Add this to your .codex/config.json:

{
  "mcpServers": {
    "aivios-coding-coach": {
      "command": "npx",
      "args": ["aivios-coding-coach"]
    }
  }
}

How It Works

  1. On startup, the server fetches the knowledge index from GitHub Pages
  2. It loads Skill files and AI knowledge documents over HTTPS
  3. Your AI client discovers the 8 tools via the MCP protocol
  4. When you ask a coding question, the AI calls the relevant tool
  5. The tool returns structured guidance -- not just a block of code
